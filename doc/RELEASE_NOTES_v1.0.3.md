# MediScreen-Brain v1.0.3 🧠

> **零配置脑肿瘤检测系统** - 双击即可使用，无需安装任何依赖

---

## ✨ 核心特性

- 🎯 **高精度检测**: 基于 YOLO 深度学习模型，精准识别脑部肿瘤区域
- ⚡ **零配置启动**: 下载 EXE 文件，双击即用，无需安装 Python 或配置环境
- 📊 **多格式支持**: 支持 NIfTI (.nii/.nii.gz)、DICOM、JPG/PNG 等医学影像格式
- 🔄 **批量处理**: 文件夹级批量检测，自动生成 Excel/TXT 报告
- 📹 **实时监控**: 多摄像头同步监控，自动录制异常片段
- 📝 **报告导出**: 一键生成 PDF 诊断报告和 Excel 统计数据

---

## 🚀 快速开始

### Windows 用户（推荐）

1. **下载**: 点击上方的 `MediScreen-Brain_1.0.3.exe` 下载
2. **运行**: 双击 EXE 文件即可启动
3. **使用**: 选择模型 → 加载图像 → 开始检测

**无需安装 Python、无需配置环境变量、无需下载依赖库！**

### 其他平台

如需在 macOS/Linux 上运行，请从源码启动：

```bash
git clone https://github.com/JingW-ui/MediScreen-Brain.git
cd MediScreen-Brain
pip install -r requirements.txt
python Brain_Tumor_detection_ui.py
```

## 📸 功能预览

### 实时检测
- 单张图像/视频/摄像头实时分析
- 三平面（轴位/矢状位/冠状位）NIfTI 医学影像分析
- 肿瘤直径、面积、体积自动测量

### 批量处理
- 自动扫描文件夹内所有图像
- 生成带标注的结果图片
- 导出 Excel 统计报表（含检测数量、置信度、尺寸等）

### 实时监控
- 最多支持 4 路摄像头同时监控
- 检测到目标时自动录制 MP4 视频
- 历史快照回放与管理
