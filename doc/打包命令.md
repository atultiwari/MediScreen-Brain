# PyInstaller 单文件打包命令

## 📦 完整打包命令（复制即用）

### 单文件模式（推荐用于分发）

```powershell
python -m PyInstaller --onefile --windowed --name="MediScreen-Brain_1.0.3" --icon="img/logo.ico" --add-data "img;img" --add-data "pt_models;pt_models" --add-data "onnx_models;onnx_models"  --hidden-import=torch  --clean --noconfirm Brain_Tumor_detection_ui.py
```

### 单文件夹模式（推荐用于开发调试）

```powershell
python -m PyInstaller --onedir --windowed --name="MediScreen-Brain" --icon="img/logo.ico" --add-data "img;img" --add-data "pt_models;pt_models" --add-data "onnx_models;onnx_models" --add-data "dcm2niix.exe;." --hidden-import=torch --hidden-import=nibabel --hidden-import=numpy --hidden-import=PySide6 --collect-submodules=ultralytics --clean --noconfirm Brain_Tumor_detection_ui.py
```
